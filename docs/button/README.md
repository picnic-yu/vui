# 按钮

按钮类型

<ClientOnly>
  <button-type/>
</ClientOnly>

按钮icon

<ClientOnly>
  <button-icon/>
</ClientOnly>



props
| 属性           | 说明                                                              | 类型  |默认值|
| -------------  |:-------------:                                                   | -----:|    -----: |
| type           | 按钮类型，可选值为 primary、info、success、warning、error或者不设置 | String |       ''   |
| icon      | 设置按钮的图标类型                                                         |   String |      ''    |
| position  | 设置按钮图标位置 可选值为 left、right、                       |    String |      'left'     |
| loading  | 设置loading、                       |    Boolean |      false     |
